# xhash

Fast and lightweight hash library that serves OpenSSL assembly optimised hash
implementation as a standalone package.

## Features

Fast, small, and standalone.

## Benchmarks

| algorithm | speed     |
|-----------|-----------|
| SHA1      | 1.50 GB/s |
